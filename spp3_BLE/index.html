BLE 指令合輯

送出方式

在「進階文字指令」輸入指令後按 Send Text。newline 可留空，Web 端會自動補上換行。

常用開機時間設定

指令
用途
範例
SET,MONITOR,<ms>
設定 Monitor 開機充氣時間。
set,monitor,15000, SET,NECK,<ms>
設定 Neck 開機充氣時間。
set,neck,12000, SET,HEAD,<ms>
設定 Head 開機充氣時間。
set,head,32000, RESET
非 Manual 狀態下回到 DRAIN_ALL，重新跑開機流程。
reset,

前三行會立即更新參數；若要讓新時間套用到開機流程，通常再送 reset,。

註：set,neck,<ms> 建議大於 10000（10 秒）；set,head,<ms> 建議大於 30000（30 秒）。

用途

ESP32 Manual 控制

指令
用途
範例
MANUAL,ENTER
進入 ESP32 Manual 控制。
MANUAL,ENTER MANUAL,STOP
停止所有輸出，保留 Manual 狀態。
MANUAL,STOP MANUAL,FILL,<MONITOR|NECK|HEAD>
單顆氣囊充氣。
MANUAL,FILL,HEAD MANUAL,DRAIN,<MONITOR|NECK|HEAD|ALL>
單顆或全部吸氣/洩氣。
MANUAL,DRAIN,ALL MANUAL,STARTUP,<head_cm>,<neck_cm>
離開 Manual 並回開機流程。
MANUAL,STARTUP,7.0,10.0

在 Manual 狀態下，舊式 reset, 只會停止輸出並停留在 Manual；要回開機流程請用 MANUAL,STARTUP,...。
